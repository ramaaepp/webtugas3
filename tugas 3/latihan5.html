Apa itu CSS?

CSS atau Cascading Style Sheet adalah dokumen web yang memiliki fungs mengatur tampilan dokumen web lain,yaitu HTML. Beberapa contoh pemanfaatan CSS pada HTML adalah untuk mengatur warna latar belakang, gaya huruf,jarak antar elemen, dan masih banyak lagi.
CSS terdiri atas kumpulan kiode-kode yang keberadanya tidak menggantikan HTML melainkan sebagai perlengkap agar tampilan dokumen HTML menjadi lebih indah dan dinamis. berikut ini adalah contoh dokumen

Bagaimana menggunakan CSS?

Ada 3 cara menggunakan CSS:

  1. Inline Styles: Menuliskan kode CSS langsung pada tag yang akan diubah atau diatur style-nya
    3. Internal/Embedded Styles: Menuliskan kode CSS apad bangian header dokumen HTML
  3. Eksternal/Linked Styles: Menuliskan kode CSS pada file terpisah dan dipanggil melalui tag link dari dokumen HTML
